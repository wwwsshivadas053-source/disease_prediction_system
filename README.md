# disease_prediction_system

MedVision AI is a deep learning based skin disease detection system built using Flask and TensorFlow. The application allows users to upload skin images and receive AI-powered predictions, confidence scores, disease information, treatment suggestions, and downloadable medical reports.

## Features

- Skin disease classification using CNN model
- Image upload and prediction
- Confidence score display
- Disease information and explanations
- Treatment and prevention suggestions
- PDF report generation
- User authentication system
- Interactive dashboard

## Technologies Used

- Python
- Flask
- TensorFlow / Keras
- SQLite
- OpenCV
- NumPy
- Pandas
- HTML, CSS, JavaScript
